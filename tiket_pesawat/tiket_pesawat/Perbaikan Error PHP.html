1. Error:

File koneksi.php tidak ditemukan pada path ../config/koneksi.php yang kamu coba include.

Solusi:

  • File koneksi.php memang ada di folder config

  • Folder config berada di satu level di atas file login.php

C:\xampp\htdocs\4SIA\Louis_003\login.php
Maka file koneksi.php harus ada di:
C:\xampp\htdocs\4SIA\config\koneksi.php

Kalau kamu meletakkan koneksi.php di folder yang berbeda, sesuaikan path-nya. Misal:

php
include('config/koneksi.php'); // jika config ada dalam folder yang sama

php

Karena variabel $conn tidak terdefinisi (lihat poin sebelumnya), maka $conn->query() menyebabkan fatal error.

Solusi:

Jika masalah koneksi database sudah teratasi (lihat poin 3), maka error ini akan hilang sendiri.
